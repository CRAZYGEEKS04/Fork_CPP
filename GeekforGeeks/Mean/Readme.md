Given the marks of N students in an Array A, calculate the mean.

Example 1:

Input:

N = 4 

A = { 56 , 67 , 30 , 79 }

Output:

58

Explanation:

56+67+30+79 = 232;  232/4 = 58.

So, the Output is 58.

Example 2:

Input:

N = 4 

A = { 78 , 89 , 67 , 76 }

Output:

77

Explanation:

78+89+67+76  = 310;  310/4 = 77.5 = 77.

So, the Output is 77.
